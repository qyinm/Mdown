# Mdown

Save web pages as clean Markdown files with one click.

## Features

- Extract article content using Mozilla Readability
- Convert HTML to Markdown with Turndown
- Download as `.md` file locally
- Works with Obsidian, Notion, Logseq and other PKM tools

## Development

```bash
pnpm install
pnpm dev        # Chrome
pnpm dev:firefox
```

## Build

```bash
pnpm build
pnpm zip        # Package for distribution
```
